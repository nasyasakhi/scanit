
## QR Scanner App

A Mobile-based QR Scanner and Generator App.



## Features

- QR Scanner Page
- QR Generator Page


## Tech Stack

**Front-End:** 

- Dart v.3.6.1 (stable)
- Flutter v.3.27.2 (stable channel)

**Dependencies:** 

- mobile_scanner
- pretty_qr_code
- permission_handler  
- share_plus  
- screenshot  
- cupertino_icons


## Usage

Needs user's permission to allow their camera.
